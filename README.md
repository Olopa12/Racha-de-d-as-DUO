🗝️ Racha de Registro de Llaves

Pequeña aplicación web tipo “Duolingo streak” creada para registrar hábitos diarios de forma sencilla y visual.

El proyecto simula un sistema de racha donde el usuario puede “fichar” una vez al día para mantener una cadena de continuidad. Si se mantiene la constancia, la racha aumenta; si se rompe, se reinicia.

✨ Características:
- Sistema de rachas (streak system)
- Almacenamiento local con localStorage
- Interfaz visual tipo app móvil
- Mascota/elemento visual dinámico según estado
- Feedback inmediato al usuario
- Diseño responsive y estilo inspirado en apps gamificadas

🧠 Objetivo:
Practicar lógica de JavaScript, manipulación del DOM, persistencia de datos y diseño de interfaces interactivas.

💡 Tecnologías usadas:
- HTML5
- CSS3
- JavaScript (Vanilla)

🚀 Posibles mejoras futuras:
- Sistema de logros
- Animaciones tipo Duolingo
- Sonidos y vibración móvil
- Niveles de usuario y estadísticas
